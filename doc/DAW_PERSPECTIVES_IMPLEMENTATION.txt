SUPPORT & TROUBLESHOOTING
================================================================================

Issue: Perspectives not showing in chat
Fix:
  1. Refresh browser (Cmd+Shift+R or Ctrl+Shift+R)
  2. Check backend is running: http://localhost:8000/docs
  3. Verify perspective markers in response: **mix_engineering**: [response]
  4. Check browser console for JavaScript errors

Issue: Wrong emoji displaying
Fix:
  1. Update browser (emoji support varies)
  2. Check font supports audio/technical emojis
  3. Try copying emoji from DAW_PERSPECTIVES_GUIDE.md

Issue: Old perspectives still showing
Fix:
  1. Clear browser cache: Cmd+K or Ctrl+K in DevTools
  2. Restart backend server
  3. Clear Codette chat history (button in UI)

Issue: Backend not recognizing new perspectives
Fix:
  1. Restart Python server: Stop & run again
  2. Check for import errors in terminal
  3. Verify codette_server_unified.py updated correctly
