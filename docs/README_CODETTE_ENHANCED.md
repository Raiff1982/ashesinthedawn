## ?? HOW IT WORKS

### The Learning Loop
```
1. User asks question
   ?
2. System shows 5 perspective responses
   ?
3. User rates response (0-4 scale)
   ?
4. System records feedback + updates preferences
   ?
5. Next question: Perspectives ordered by user preference
   ?
6. User sees MORE personalized responses!
```

### The Algorithm
```
Preference Update:
  new_score = (old_score × 0.7) + (rating_influence × 0.3)

Effect:
  - Week 1: Generic perspectives (all 0.5)
  - User rates Mix Engineering 4/4
    ? new_score = 0.65 (above average)
  - User rates Audio Theory 1/4
    ? new_score = 0.425 (below average)
  - Week 2: Same question
    ? Mix Engineering appears FIRST (preferred)
    ? Audio Theory appears LAST (less preferred)
  - Result: PERSONALIZED response!
```
